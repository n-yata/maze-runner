# パックマン風ゲーム - アイデアメモ

## コンセプト

GitHub Pages で無料公開する、スマホでも遊べるパックマン風ゲーム。
著作権への配慮から、タイトル・素材はオリジナルで制作する。

---

## 技術スタック

| 項目 | 選定 | 理由 |
|------|------|------|
| 言語 | Vanilla JS + Canvas API | ビルド不要、依存ゼロ、GitHub Pages に git push だけでOK |
| ライブラリ | なし | 完全制御できるため、忠実再現に最適 |
| ホスティング | GitHub Pages | 無料、静的ファイル配信 |

---

## 操作仕様（モバイル）

- **スワイプ操作**を採用（Dパッドは不採用）
- スワイプした瞬間に**即時反応**
- 壁がある方向にスワイプ → **その位置でピタッと停止**（バッファリングなし）
- スワイプ検知ロジック：
  - `touchstart` で開始座標を記録
  - `touchend` で差分ベクトルを計算
  - X差分 vs Y差分の大きい方を意図した方向と判定
  - 閾値（例: 30px）以下は無視（誤検知防止）
  - `preventDefault()` でページスクロールを抑制

## ゲーム仕様

### フェーズ1（初期リリース）
- 原作1面のマップを忠実に再現
- ドット・パワーエサの配置も原作準拠
- スコアシステム

### フェーズ2（次のステップ）
- レベル進行（面クリアで次の面へ）
- 難易度上昇（ゴーストの速度など）

### ゴーストAI（原作準拠・4体）

| 名前 | 色 | 行動パターン |
|------|----|-------------|
| Blinky | 赤 | プレイヤーを直接追跡 |
| Pinky | ピンク | プレイヤーの進行方向4マス先を狙う |
| Inky | 青 | BlinkyとPinkyを組み合わせた複雑な挟み撃ち |
| Clyde | オレンジ | 近づくと逃げる、遠いと追跡 |

## PWA対応

スマホで「ホーム画面に追加」してアプリとして起動できるようにする。

| ファイル | 役割 |
|----------|------|
| `manifest.json` | アプリ名・アイコン・テーマ色の定義 |
| `service-worker.js` | オフラインキャッシュ、インストール対応 |

---

## 著作権への対応

| 項目 | 方針 |
|------|------|
| タイトル名 | 「PAC-MAN」は商標のため使用しない。「迷宮ラン」を使用する |
| キャラクター素材 | オリジナルのドット絵を自作（似ていてもOK、コピーはNG） |
| 音源・SE | BGM・SEあり。フリー素材を活用 |
| ゲームメカニクス | 著作権保護の対象外のため再現OK |
| 公開形態 | 非商用（GitHub Pages 無料公開） |

> ゲームのルール・メカニクス自体は著作権で保護されない。
> ただし商標名・オリジナル素材の無断使用はリスクあり。
